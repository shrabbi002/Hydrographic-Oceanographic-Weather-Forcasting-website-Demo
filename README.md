
# Hydrographic Oceanographic Weather Forecasting Website Demo

This is a demo website project for hydrographic, oceanographic, and weather forecasting, built with **Next.js** and **TypeScript**. The project showcases a web interface that includes multiple pages, such as Products & Services sections. Currently, it is in the early development stages with basic navigation components and demo data integration.

## Features

- Hydrographic and oceanographic data forecasting interface
- Weather forecasting website demonstration
- Multi-page navigation system with dropdown menus
- Products and Services pages with demo data
- Responsive web components
- Environment configuration support

## Technology Stack

- **Frontend Framework:** Next.js (React)
- **Language:** TypeScript (97.3%)
- **Styling:** CSS (1.5%), PostCSS
- **Package Manager:** npm/pnpm
- **Database Support:** PLpgSQL (1.1%)
- **Build Tool:** Webpack
- **Configuration Files:** 
  - `tsconfig.json`
  - `next.config.mjs`
